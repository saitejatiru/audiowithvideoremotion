# Plan: 03-02 - Prompter and Client

**Requirement(s):** STORY-01

**Context:**
We need to generate the system prompt containing the JSON schema and the client to call OpenAI-compatible APIs (DeepSeek, Minimax, Kimi).

**Instructions:**
1. Create `storyboard/prompter.py`.
2. Implement `build_system_prompt()` that calls `LLMScenesResponse.model_json_schema()` and returns the instruction string (enforcing JSON schema, 1 scene per sentence).
3. Implement `build_user_prompt(sentences)` that formats the sentences into a numbered list.
4. Create `storyboard/client.py`.
5. Implement `call_llm(messages, base_url, api_key, model)` using the `openai` SDK. Note: import `openai` inside the function body to defer import overhead. Use `response_format={"type": "json_object"}`. Handle the empty content edge case.
6. Create `storyboard/tests/test_prompter.py` to ensure schema is embedded properly.

**Acceptance Criteria:**
- System prompt contains the string `"visual_type"` and the schema structure.
- Client function handles empty returns and passes `response_format` correctly.
