# Plan: 05-01 - Metadata Stripping & Web Faststart

**Requirement(s):** POST-01, POST-02

**Context:**
The rendered video from Remotion needs to be post-processed to remove metadata and fingerprints, and optimized for web streaming by moving the `moov` atom to the beginning of the file. We will invoke `ffmpeg` in a subprocess.

**Instructions:**
1. Create `post_process/` package structure:
   - `post_process/__init__.py`
   - `post_process/post_processor.py`
   - `post_process/tests/__init__.py`
   - `post_process/tests/test_post_processor.py`
2. Implement `post_process_video(input_path: str, output_path: str) -> str`:
   - Builds and runs the `ffmpeg` command: `ffmpeg -y -i <input> -map_metadata -1 -c copy -movflags +faststart -fflags +bitexact <output>`
   - Handles subprocess error codes, raising `PostProcessError` on failure.
   - Defer `ffmpeg` execution to runtime subprocess calls.
3. Implement a helper to verify that no metadata tags remain using `ffprobe`:
   - Run `ffprobe -v quiet -print_format json -show_format <output_path>`
   - Check that `format.tags` is empty or missing.
4. Implement unit tests in `test_post_processor.py`:
   - Mock `subprocess.run` to test command args construction and error handling.
   - Mock `ffprobe` output to verify the metadata check logic.

**Acceptance Criteria:**
- Unit tests pass cleanly.
- `post_process_video` raises `PostProcessError` if `ffmpeg` fails.
