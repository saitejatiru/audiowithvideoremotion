# Plan: 04-01 - Remotion Project Scaffold

**Requirement(s):** VIDEO-01, VIDEO-03

**Context:**
Phase 4 introduces the Node.js/React side of the pipeline. Remotion renders video from
React components. We create a `video/` subdirectory with a Remotion project that reads
`timeline.json` as `inputProps`.

**Instructions:**
1. Create `video/` directory with Remotion project structure (no `npx create-video` — manual setup is cleaner for integration).
2. Create `video/package.json` with dependencies: `remotion`, `@remotion/cli`, `@remotion/bundler`, `react`, `react-dom`.
3. Create `video/tsconfig.json` for TypeScript/JSX.
4. Create `video/src/Root.tsx` — registers the `ExplainerVideo` composition with `calculateMetadata` for dynamic `durationInFrames` from `timeline.audio.durationSec`.
5. Create `video/src/index.ts` — entry point that exports `Root`.
6. Create `video/remotion.config.ts` — Remotion bundler config.
7. Verify: `cd video && npm install` completes without error.

**Acceptance Criteria:**
- `npm install` in `video/` succeeds.
- Remotion project structure is valid.
