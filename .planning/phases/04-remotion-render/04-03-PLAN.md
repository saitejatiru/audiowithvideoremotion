# Plan: 04-03 - Render Script and Python Bridge

**Requirement(s):** VIDEO-01, VIDEO-03

**Context:**
The Python orchestrator needs to call `npx remotion render` with the right props.
This plan creates the render script and the Python bridge that Phase 6 will use.

**Instructions:**
1. Create `video/render.sh` — shell script that runs `npx remotion render ExplainerVideo out.mp4 --props <timeline.json>`.
2. Create `video/src/render-cli.ts` — programmatic render entry point (alternative to CLI).
3. Create `render/render_bridge.py` — Python module that:
   - Takes a `timeline.json` path and output MP4 path
   - Calls `npx remotion render` via subprocess
   - Validates the output file exists and has correct duration (via ffprobe)
   - Returns the output path
4. Create `render/tests/test_render_bridge.py` — mock subprocess tests verifying the bridge constructs the correct command and handles errors.

**Acceptance Criteria:**
- Python bridge constructs correct remotion render command.
- Bridge handles subprocess failure gracefully.
- Duration validation logic works with mock ffprobe output.
