# Plan: 04-02 - Scene and Caption Components

**Requirement(s):** VIDEO-01, VIDEO-02, VIDEO-04

**Context:**
The visual components that render scenes and word-level captions based on the current frame.

**Instructions:**
1. Create `video/src/types.ts` — TypeScript interfaces matching `timeline.json` schema.
2. Create `video/src/components/SceneRenderer.tsx`:
   - Receives `timeline.scenes` + current time
   - Renders the active scene's `onScreenText` and visual type indicator
   - Scene transitions at sentence boundaries
3. Create `video/src/components/CaptionRenderer.tsx`:
   - Receives `timeline.words` + current time
   - Highlights the currently-spoken word (TikTok-style active word)
   - Different colors per `speaker` value (VIDEO-04)
4. Create `video/src/ExplainerVideo.tsx`:
   - Main composition component
   - Uses `<Audio>` for narration
   - Composes `SceneRenderer` + `CaptionRenderer`
   - Uses `useCurrentFrame()` + `useVideoConfig()` for time calculations

**Acceptance Criteria:**
- Components compile without TypeScript errors.
- Active scene/word filtering uses `useCurrentFrame() / fps` for time.
